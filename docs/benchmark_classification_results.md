# FLAb2 ベンチマーク分類評価レポート

## 1. 概要

FLAb2（Fitness Landscape for Antibodies, version 2）は、抗体言語モデル（AbLM）が配列表現を通じて抗体の生物学的フィットネスをどの程度捉えられるかを体系的に評価するベンチマークスイートである。本評価では5つの抗体/タンパク質言語モデルの埋め込みを抽出し、SVM と XGBoost による二値分類タスクで AUC-ROC を中心に性能を比較した。

---

## 2. ベンチマークデータセットの詳細

FLAb2 は結合親和性・発現・免疫原性・熱安定性・凝集・多反応性・薬物動態の7カテゴリにわたるデータセットを収録している。各データセットの元論文・対象抗体・抗原・測定指標を以下に記す。

### 2.1 Binding（結合親和性）

#### koenig2017_kd — 抗VEGF抗体 G6 の Kd

- **元論文**: Koenig P. et al., "Mutational landscape of antibody variable domains reveals a switch modulating the interdomain conformational dynamics and antigen binding." *PNAS*, 2017.
- **対象抗体**: G6（抗 VEGF 抗体）
- **対象抗原**: **VEGF（血管内皮増殖因子）**
- **測定**: 深部変異スキャン（DMS）により G6 の VH/VL 全単一アミノ酸変異体を作製し、酵母ディスプレイ + FACS 選別で VEGF に対する解離定数（Kd）を定量。
- **サンプル数**: 4,275（連続値を中央値で二値化）

#### kirby2024_binary — 抗 SARS-CoV-2 抗体の Kd

- **元論文**: Kirby M.B. et al., 2024
- **対象抗体**: 複数の SARS-CoV-2 中和抗体（臨床・研究段階のパネル）
- **対象抗原**: **SARS-CoV-2 スパイクタンパク質（主に RBD）**
- **測定**: 抗体の結合親和性を閾値で二値化（結合 / 非結合）。過去の報告データを統合した回顧的解析。
- **サンプル数**: 1,407（バイナリラベル）

#### peterson2024_binary — 抗インフルエンザ H1 HA 抗体の結合

- **元論文**: Peterson J.N. et al., 2024
- **対象抗体**: インフルエンザ H1 HA に対する抗体群
- **対象抗原**: **インフルエンザウイルス H1 亜型ヘマグルチニン（H1 HA）**
- **測定**: H1 HA に対する結合の有無をバイナリで評価。複数の変異体ライブラリを統合。
- **サンプル数**: 1,071（バイナリラベル）

#### warszawski2019_kd — 抗リゾチーム抗体 D44 の Kd

- **元論文**: Warszawski S. et al., "Optimizing antibody affinity and stability by the automated design of the variable light-heavy chain interfaces." *PLOS Computational Biology*, 2019.
- **対象抗体**: D44（抗リゾチーム scFv、HyHEL-10 由来）
- **対象抗原**: **リゾチーム（Lysozyme）**
- **測定**: VH–VL 界面の計算設計変異体について、リゾチームに対する Kd を実験的に測定。
- **サンプル数**: 2,048（連続値を中央値で二値化）

### 2.2 Expression（発現量）

#### koenig2017_er — 抗 VEGF 抗体 G6 の発現率

- koenig2017_kd と同一実験由来。酵母ディスプレイにおける G6 変異体の表面発現効率（蛍光強度）を定量。抗体の折りたたみ・安定性の代理指標。
- **サンプル数**: 4,275

#### jain2017_hek — 治療用抗体パネルの HEK293 発現量

- **元論文**: Jain T. et al., "Biophysical properties of the clinical-stage antibody landscape." *PNAS*, 2017.
- **対象抗体**: 臨床開発段階の治療用 IgG 抗体パネル（約150種）
- **測定**: HEK293 細胞での一過性発現量（mg/L）。製造可能性の重要指標。
- **サンプル数**: 137

### 2.3 Immunogenicity（免疫原性）

#### marks2021_immuno — ヒト化抗体の免疫原性

- **元論文**: Marks C. et al., 2021
- **対象抗体**: マウス由来フレームワークのヒト化ライブラリ
- **測定**: T 細胞エピトープ予測または実験的アッセイによる免疫原性スコア。ヒト化程度と免疫原性リスクの関係を定量。
- **サンプル数**: 217

### 2.4 Thermostability（熱安定性）

#### jain2017_tm — 治療用抗体パネルの融解温度

- Jain 2017 と同一論文。示差走査蛍光測定（DSF）による Tm を測定。開発適性の重要指標。
- **サンプル数**: 137

### 2.5 Aggregation（凝集性）

#### jain2017_hicrt — 治療用抗体パネルの HIC 保持時間

- Jain 2017 と同一論文。疎水性相互作用クロマトグラフィー（HIC）の保持時間を測定。抗体表面疎水性の代理指標であり、凝集リスクと強く相関する。
- **サンプル数**: 137

### 2.6 Polyreactivity（多反応性）

#### jain2017_psr — 治療用抗体パネルの多反応性スコア

- Jain 2017 と同一論文。バイオチン化した非特異的抗原カクテル（ソルビトールデヒドロゲナーゼ、インスリン等）に対する ELISA 結合スコア（PSR）。
- **サンプル数**: 137
- **注記**: 中央値二値化の結果、全サンプルが同一クラスに分類されたため**評価対象外**とした。

### 2.7 Pharmacokinetics（薬物動態）

#### jain2023_fcrn — FcRn 結合相対保持時間

- **元論文**: Jain T. et al., 2023
- **測定**: FcRn（新生児型 Fc 受容体）への pH 6.0 でのアフィニティークロマトグラフィー保持時間。FcRn 結合は IgG の血清半減期の主要決定因子であり、薬物動態と直結する。
- **サンプル数**: 132

## 3. 実験設定

### 3.1 モデル

| モデル | 埋め込み次元 | 事前学習データ | 学習目標 |
|--------|------------|-------------|---------|
| AntiBERTy | 512 | OAS（ヒトペア抗体） | MLM |
| AbLang | 768 | OAS（重鎖） | MLM |
| AbLang-2 (heavy-only) | 480 | OAS（ペア抗体） | MLM |
| ESM-2 (650M) | 1280 | UniRef50（汎用タンパク質） | MLM |
| ProGen-2 (medium) | 1536 | UniRef + BFD 等（汎用タンパク質） | 自己回帰（次トークン予測） |

**重要**: 本ベンチマークでは**全モデルで VH（重鎖可変領域）配列のみ**を入力とした。FLAb2 の CSV ファイルの `heavy` カラムのみを使用し、軽鎖は使用していない。AntiBERTy および AbLang-2 はペア配列対応モデルだが、本実験では重鎖単鎖のみを入力している。

### 3.2 プーリング

- **seq**: VH 全残基の隠れ状態ベクトルの平均
- **cdr3**: CDR-H3 残基に対応する隠れ状態ベクトルの平均（`abnumber` でインデックス抽出）
- いずれも**全長配列を入力**してから対象残基の出力をプーリングする。CDR-H3 配列を単独入力しているわけではない。

### 3.3 分類器

- **SVM**: `StandardScaler` + `SVC(kernel='rbf', probability=True, random_state=42)`
- **XGBoost**: `XGBClassifier(n_estimators=100, max_depth=6, learning_rate=0.1, random_state=42)`

### 3.4 評価プロトコル

- 5-fold 層化交差検証（`StratifiedKFold, shuffle=True, random_state=42`）
- 評価指標: Accuracy、AUC-ROC、F1 macro（平均 ± 標準偏差）
- 連続値ラベルは中央値で二値化（中央値以上 → 1、未満 → 0）

## 5. 考察

### 5.1 全体的な傾向: ESM-2 と AbLang-2 が平均上位だが、解釈には注意が必要

10データセット平均では ESM-2（SVM seq: 0.695）と AbLang-2（SVM cdr3: 0.690）が上位を占めた。しかし、この結果からモデルの表現能力を直接比較するには以下の交絡因子を考慮する必要がある。

**埋め込み次元数の差**: ESM-2（1280d）および ProGen-2（1536d）は抗体特化モデル（480〜768d）の2〜3倍の次元を持つ。RBF SVM + StandardScaler の組み合わせでは、高次元空間の方が分離余地が広がる場合があり、性能差が「モデルの表現品質」と「次元数の効果」のどちらに起因するかを本実験のみで区別することはできない。次元の影響を排除するには、PCA 等で同一次元に揃えた上で再評価する追加実験が望ましい。

**データセット構成の偏り**: ESM-2 の全体平均は peterson2024_binary（AUC = 0.905）に大きく引き上げられている。この1データセットを除外すると ESM-2 の平均は 0.672 まで低下し、AbLang-2（0.666）や AbLang（0.662）との差は極めて小さくなる。

これらの留保を踏まえると、「汎用モデルが抗体特化モデルを一貫して上回った」と結論づけるのは時期尚早であり、タスク・データセットごとの性質に応じたモデル選択が重要である。

### 5.2 ProGen-2 の高分散と thermostability での系統的誤分類

ProGen-2 は全モデル中で最大の標準偏差（SVM seq: ±0.113）を示し、kirby2024_binary（0.788）や peterson2024_binary（0.846）では最上位クラスの性能を示す一方、jain2017_tm（thermostability）では AUC = 0.469 とランダム（0.5）を下回った。

AUC < 0.5 は「予測に情報がない」のではなく、**系統的に逆方向の予測をしている**ことを意味する。ProGen-2 の埋め込み空間において、熱安定クラスと非安定クラスの位置関係が分類器の期待と逆転している可能性がある。自己回帰モデルの「生成しやすさ」が高い配列が必ずしも熱安定性の高い配列とは限らず、むしろ一般的なレパートリー配列を好む学習バイアスが安定性シグナルと逆相関している可能性が考えられる。また、jain2017_tm は N = 137 と小規模であり、CV の分散が大きいことも影響し得る。この現象の原因特定には、ProGen-2 埋め込みの PCA / UMAP 可視化によるクラス分離構造の確認が必要である。

### 5.3 Binding カテゴリの一貫した高性能

Binding カテゴリ（4データセット）は全モデルで最も高い AUC-ROC を示した（カテゴリ平均 0.71〜0.75）。特に peterson2024_binary（H1 HA 結合）では ESM-2 が AUC = 0.905、AbLang-2 が 0.868 と突出した性能を記録した。

4つの binding データセットはそれぞれ異なる抗原（VEGF, SARS-CoV-2 RBD, インフルエンザ H1 HA, リゾチーム）を対象としており、単一抗原への過適合では説明できない。これは抗体配列埋め込みが、特定の抗原に対する結合親和性ではなく、「結合能を持つ配列」と「結合能を持たない配列」を区別する一般的な配列特徴（CDR 領域の長さ・アミノ酸組成・電荷分布など）を捉えていることを示唆する。

### 5.4 CDR3 プーリングと seq プーリングの差が小さい理由

CDR3 プーリングと seq プーリングの AUC-ROC 差は大半の組み合わせで ±0.05 以内であった。ただし、この結果の解釈には方法論的な留意点がある。

本実験の CDR3 プーリングは、**全長 VH 配列を入力**した上でモデルが出力した隠れ状態のうち CDR-H3 残基に対応するベクトルを取り出している。Transformer ベースのモデルではアテンション機構を通じて全残基間の情報伝播が行われるため、CDR-H3 位置の出力ベクトルにはフレームワーク領域の文脈情報が混入している。したがって、「CDR3 プーリングで性能が落ちない」ことから「CDR-H3 のみで十分」とも「フレームワーク領域が不要」とも結論できない。

CDR-H3 の情報だけで十分かどうかを厳密に検証するには、CDR-H3 配列を単独入力した場合の性能と比較する必要がある。

### 5.5 開発適性指標（Developability）の予測困難さ

immunogenicity（marks2021_immuno, 最高 AUC = 0.640）、aggregation（jain2017_hicrt, 最高 0.682）、thermostability（jain2017_tm, 最高 0.687）は全モデルで AUC < 0.70 にとどまった。これらの開発適性指標は以下の理由から配列埋め込みのみでの予測が困難と考えられる。

- **免疫原性**: MHC-II へのペプチド提示効率、T 細胞レパートリー、ヒト化程度など、配列以外の宿主免疫系との相互作用に大きく依存する。
- **凝集性**: 溶液中の三次元構造における疎水性パッチの表面露出（SP3D）が主要因であり、配列からの間接的推定にはおのずと限界がある。
- **熱安定性**: フレームワーク内のパッキング密度、VH-VL 界面の相互作用、ジスルフィド結合の形成などの立体構造要因が支配的である。

ただし、これらのデータセットはいずれも N = 137〜217 と小規模であり、5-fold CV の各 fold の学習サンプル数は 100〜170 程度にとどまる。統計的検出力の不足やラベルの実験間ばらつき（特に immunogenicity はアッセイ依存性が高い）も低 AUC に寄与している可能性があり、「モデルの限界」と断言する前にデータ充足性の検討が必要である。

### 5.6 SVM vs XGBoost: データセット規模との関係

全体平均では SVM と XGBoost の差は小さい（0.01〜0.02 程度）が、データセット規模による傾向がみられた。

小規模データセット（N < 200）のうち、jain2023_fcrn（N = 132）では全5モデルで SVM が XGBoost を上回った（例: AbLang SVM 0.722 vs XGBoost 0.666）。jain2017_hicrt でも AntiBERTy（SVM 0.682 vs XGBoost 0.571）・ESM-2（SVM 0.674 vs XGBoost 0.619）で SVM が優位であった。一方、jain2017_hek では逆に XGBoost が SVM を上回るモデルが多かった。

高次元（480〜1536d）かつ小サンプル（N < 200）の設定では、RBF SVM のマージン最大化と正則化（パラメータ C）が過学習を抑制しやすい一方、XGBoost はブースティングの積み重ねにより過学習リスクが高まる傾向がある。ただし本実験では XGBoost のハイパーパラメータを固定しており（`max_depth=6, n_estimators=100`）、チューニングにより結果が変わる可能性は否定できない。

### 5.7 抗体特化モデルのタスク特異的な強み

抗体特化モデル（AntiBERTy, AbLang, AbLang-2）は全体平均では ESM-2 にわずかに劣るものの、特定タスクで最上位の性能を示した。

- **AbLang**: pharmacokinetics（jain2023_fcrn, AUC = 0.722）で全モデル中最高。FcRn 結合に関連する Fc 近傍の配列特徴を重鎖特化の事前学習で効果的に捉えている可能性がある。
- **AbLang-2**: binding カテゴリ平均（0.748）で全モデル中最高。ペア抗体コーパスで事前学習された表現が、重鎖のみ入力でも結合予測に有効に働いている。
- **AntiBERTy**: aggregation（jain2017_hicrt, AUC = 0.682）で最高。抗体配列に特化した事前学習を通じて、凝集傾向と関連する疎水性パッチ・芳香族残基密度などの配列特徴が埋め込みに反映されている可能性がある。

これらの結果は、汎用モデルと抗体特化モデルそれぞれに補完的な強みがあることを示しており、タスクに応じたモデル選択が実用上重要であることを示唆する。

## 6. まとめ

- **全体平均**: ESM-2（SVM seq: 0.695）が最高平均だが、埋め込み次元数の差（1280d vs 480〜768d）という交絡因子があり、直接的なモデル品質の比較には注意を要する。
- **タスク特異的な強み**: AbLang は pharmacokinetics、AbLang-2 は binding、AntiBERTy は aggregation で各々最上位を示した。単一モデルが全タスクで支配的ではない。
- **Binding カテゴリ**: 異なる4抗原（VEGF, SARS-CoV-2 RBD, H1 HA, リゾチーム）に対して全モデルで高い予測性能を示し、配列埋め込みが結合能を反映する一般的特徴を捉えていることを示唆。
- **開発適性指標の限界**: thermostability・immunogenicity・aggregation は全モデルで AUC < 0.70 にとどまり、配列情報のみでの予測が困難。ただしデータセット規模の小ささも寄与している可能性あり。
- **ProGen-2 の不安定性**: 分散最大（±0.113）、thermostability で AUC = 0.469（ランダム以下）を示し、タスクによっては系統的な誤分類が生じる。
- **CDR3 プーリング**: seq プーリングと概ね同等だが、全長入力の文脈情報が CDR3 ベクトルに混入しているため、CDR-H3 単独の情報量に関する結論は導けない。
